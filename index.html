Titulos <h1>,<h2> e em diante:

(Voltar)

Os títulos possuem SEIS NÍVEIS de hierarquia, e esses níveis servem para organizar nosso conteúdo. Para isso, usamos as tags h1, h2, h3, h4, h5 e h6 para demarcar nossos títulos.

Um h1 significa que esse é um assunto principal e é sobre ele que vamos escrever. Já o h2 significa que esse é um sub-assunto do h1 que está imediatamente acima dele. O h3 significa que esse é um sub-assunto do h2 que está acima dele. E assim sucessivamente até o h6.

Paragrafo - <p>:

(Voltar)

A tag <p> em HTML é usada para definir um parágrafo de texto. Ela é um elemento de marcação que indica ao navegador da web que o conteúdo dentro dessa tag deve ser tratado como um novo parágrafo. Os navegadores geralmente exibem o texto contido em uma tag <p> com espaço extra acima e abaixo, criando uma separação visual entre os parágrafos.

Marcado - <mark>:

(Voltar)

A tag <mark> em HTML é usada para destacar ou marcar parte do texto dentro de um elemento, indicando visualmente que esse trecho de texto é relevante ou possui alguma importância especial. Geralmente, o texto marcado com a tag <mark> é realçado com uma cor de fundo amarela por padrão, embora a aparência possa variar dependendo do navegador e da folha de estilo aplicada.

Tamanho de fonte menor - <small>:

(Voltar)

A tag HTML <small> é usada para definir texto que deve ser exibido em um tamanho de fonte menor do que o texto circundante. Ela é frequentemente usada para indicar informações secundárias, notas de rodapé, avisos legais ou outras informações menos relevantes em relação ao conteúdo principal.

Para dar ênfase - <i>:

(Voltar)

A tag <i> costumava ser usada para itálico, mas agora é mais comumente usada para indicar ênfase no texto, mesmo que isso não signifique necessariamente itálico. Ajuda a destacar palavras sem alterar o significado.

Para sublinhar texto - <u>:

(Voltar)

A tag <u> costumava sublinhar o texto, mas hoje em dia é raramente usada, já que a sublinhagem é normalmente associada a links. Pode ser melhor usar CSS para aplicar estilos visuais.

Negrito - <strong>:

(Voltar)

A tag <strong> é usada para enfatizar um trecho de texto de maneira mais forte, geralmente exibido em negrito. Indica importância ao leitor e tem significado semântico.

Lista ordenada - <ol>:

(Voltar)

A tag <ol> cria uma lista ordenada, onde cada item é automaticamente numerado. É útil quando você deseja listar itens em uma sequência específica, como instruções passo a passo.

Lista não ordenada - <ul>:

(Voltar)

A tag <ul> cria uma lista não ordenada, onde cada item é precedido por um marcador (geralmente um ponto). É ideal para listas sem uma ordem específica.

Item de uma lista - <li>:

(Voltar)

A tag <li> define um item em uma lista, seja ordenada (<ol>) ou não ordenada (<ul>). Cada <li> é um elemento individual da lista.

Âncora de links - <a>:

(Voltar)

A tag <a> é usada para criar links âncora, permitindo que você conecte uma página da web a outra, ou a um recurso externo, como um arquivo ou e-mail.

Linha Horizontal - <hr>:

(Voltar)

A tag <hr> insere uma linha horizontal no documento, geralmente usada para separar seções de conteúdo.

Texto subscrito - <sub>:

(Voltar)

A tag <sub> coloca o texto como subscrito, o que significa que ele aparecerá um pouco abaixo da linha de base do texto normal, como em fórmulas químicas.

Texto sobreescrito - <sup>:

(Voltar)

A tag <sup> coloca o texto como sobrescrito, o que significa que ele aparecerá um pouco acima da linha de base do texto normal, como em expoentes.

Citação longa - <blockquote>:

(Voltar)

A tag <blockquote> é usada para criar um bloco de citação, normalmente recuado, para destacar uma citação ou um trecho longo de texto.

Fonte - <font>:

(Voltar)

Embora já tenha sido usada para alterar a fonte e outros estilos de texto, é agora desencorajada em favor de CSS para controlar a aparência visual do texto.

Texto deletado - <del>:

(Voltar)

A tag <del> é usada para mostrar texto como riscado ou deletado, indicando que algo foi removido ou alterado.

Abreviação - <abbr>:

(Voltar)

A tag <abbr> é usada para definir uma abreviação ou acrônimo e, muitas vezes, exibe um tooltip quando o cursor do mouse é posicionado sobre ele, mostrando a descrição completa da abreviação.
